## About

Travel management system in two versions: an HTML/CSS/JS prototype for layout and flows, and a MEAN stack application with Angular and MongoDB. The V2 app was not fully completed; login and register were implemented.

## Features

- **Version 1 (HTML, CSS, JS)**
  - Static prototype to validate layout and user flows for the travel system
  - Demo hosted on GitHub Pages for sharing

- **Version 2 (MEAN stack — Angular, Node.js, MongoDB)**
  - **Login** — user authentication flow implemented
  - **Register** — user registration flow implemented
  - Rest of the application (travel booking, management, etc.) was not completed

## Implementation history

- **Travel-V1**: Static prototype in HTML, CSS, and JavaScript to validate the layout and flows for the travel management concept.
- **Travel-V2**: MEAN stack application (MongoDB, Express, Angular, Node.js) with Angular frontend and MongoDB backend. Only the login and register functionality was completed; the full travel management features were not built out.
