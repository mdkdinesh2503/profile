## Overview

A personal portfolio site built during college as a self-directed project. Designed and implemented from scratch to practice front-end fundamentals and express my own approach to layout, structure, and presentation.

## What I did

- Designed and built the layout and pages from own thoughts.
- Used HTML, CSS, and JavaScript for structure, styling, and interactivity.
- Hosted as a GitHub Pages demo for sharing and reference.

## Takeaways

- Learned to plan and implement a multi-page portfolio without a framework.
- Gained practice in responsive layout and clean, maintainable markup.
